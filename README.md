# Huntd Manual Testing Project (Jira + TestRail)

---

## 📌 Project Overview

This repository contains a manual QA testing project for the Huntd application.

Testing was performed as part of a team project and covered:
- web application (desktop browser)
- mobile web browser
- mobile application (Android)

---

## 🎯 Scope of Work

The project included:

- test plan creation
- functional decomposition (web & mobile)
- permission testing table
- requirements traceability matrix (RTM)
- test case design (TestRail)
- bug reporting (Jira)
- exploratory testing
- test execution and reporting

---

## 🧪 What Was Tested

### Core Features
- authentication (sign in / sign up)
- profile management
- job search and filters
- role switching (Candidate / Recruiter)
- chat functionality
- job posting
- navigation and UI elements

---

## 🛠 Tools Used

- Jira (bug tracking, task management)
- TestRail (test case management – documented)
- Chrome DevTools
- Android device
- GitHub

## 📂 Repository Structure

- `/docs` → test plan, scope, decomposition, reports
