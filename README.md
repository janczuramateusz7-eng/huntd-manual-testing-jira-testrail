# Huntd Manual Testing Project (Jira + TestRail)

This repository contains my manual QA testing project for the Huntd application.

---

## Project Overview

The project covered manual testing activities for:
- web application
- mobile web browser
- mobile application

Testing was performed as part of a team project using Jira and TestRail.

---

## Scope of Work

The project included:
- test plan preparation
- decomposition for web and mobile applications
- Jira + TestRail integration
- permission testing table
- RTM creation
- task prioritization and distribution
- story creation in Jira
- test case design in TestRail
- bug reporting
- final test report and execution summary

---

## Tools Used

- Jira
- TestRail
- Chrome DevTools
- Mobile emulator / real device
- GitHub

## Repository Structure
