# Project Scope – Huntd Application

---

## 📌 Overview

This project covers manual testing of the Huntd application across multiple platforms:
- web application (desktop browser)
- mobile application (Android)

The goal is to verify that the application works correctly and meets functional requirements.

---

## 🎯 Scope of Testing

### Platforms
- Web (desktop)
- Mobile application (Android)

---

## 🔍 Functional Areas

The following features were tested:

### Authentication
- sign in / sign up
- social login
- logout

### Profile
- profile view
- profile editing
- role switching (Candidate / Recruiter)

### Jobs
- job offers list
- job details
- job posting

### Candidates
- candidate list
- candidate profile
- interactions

### Filters
- filters visibility
- filters functionality
- restrictions for unauthorized users

### Chats
- all chats
- archive chats

### Other
- homepage content
- navigation
- footer and static sections

---

## ❌ Out of Scope

- iOS mobile application
- admin functionality
- performance testing
- security testing
- backend/API testing

---

## 🎯 Goal

The goal of this testing is to ensure that:
- core functionalities work correctly
- user flows are stable
- application behavior is consistent across platforms
