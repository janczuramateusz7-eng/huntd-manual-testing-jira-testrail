# TestRail Structure

---

## 📌 Overview

TestRail was used to organize and manage test cases for both web and mobile applications.

---

## 🧩 Project Structure

The project was divided into two main areas:

### Web Application
- Homepage
- Authentication
- Profile
- Candidates
- Filters
- Chat
- Jobs

### Mobile Application
- Authentication
- Profile
- Job browsing
- Chat
- Navigation

---

## 🧪 Test Case Design

Test cases included:
- positive scenarios
- negative scenarios
- validation checks
- permission-based scenarios

Test cases were executed and results were tracked during testing.
