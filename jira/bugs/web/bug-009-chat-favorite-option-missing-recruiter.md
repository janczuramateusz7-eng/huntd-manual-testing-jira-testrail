# Bug 009 – "Add to Favorite" option missing in chat for Recruiter

---

## 📌 Summary
The "Add to Favorite" option is missing in chat functionality for Recruiter users.

---

## 🧪 Environment
- Application: Huntd (Web)
- Browser: Chrome 145.0.7632.76

---

## 🔁 Steps to Reproduce
1. Open the main page
2. Log in to the application
3. Switch account to Recruiter role
4. Navigate to the Chats page
5. Open any chat conversation

---

## ❌ Actual Result
There is no option or icon available to mark the chat as favorite.

---

## ✅ Expected Result
Recruiter user should be able to mark chat as favorite (e.g., via icon or button).

---

## ⚠️ Severity
Medium

---

## 🚨 Priority
Medium
