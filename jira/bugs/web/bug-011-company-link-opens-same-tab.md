# Bug 011 – Company link opens in the same tab instead of a new tab

---

## 📌 Summary
Company link opens in the same browser tab instead of a new tab.

---

## 🧪 Environment
- Application: Huntd (Web)
- Browser: Chrome 145.0.7632.76

---

## 🔁 Steps to Reproduce
1. Open the main page
2. Scroll down to the footer section
3. Click on any company logo

---

## ❌ Actual Result
Company vacancies page opens in the same browser tab.

---

## ✅ Expected Result
Company vacancies page should open in a new browser tab.
