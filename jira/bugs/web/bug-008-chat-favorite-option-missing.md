# Bug 008 – "Add to Favorite" option missing in chat

---

## 📌 Summary
The "Add to Favorite" option is missing in the chat functionality.

---

## 🧪 Environment
- Application: Huntd (Web)
- Browser: Chrome 145.0.7632.76

---

## 🔁 Steps to Reproduce
1. Open the main page
2. Log in to the application
3. Navigate to the Chats page
4. Open any chat conversation

---

## ❌ Actual Result
There is no option or icon available to mark the chat as favorite.

---

## ✅ Expected Result
User should be able to mark chat as favorite (e.g., via icon or button).

---

## ⚠️ Severity
Medium

---

## 🚨 Priority
Medium
