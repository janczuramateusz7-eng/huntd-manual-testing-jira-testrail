# Bug 003 – Euro currency option not available in Expected Salary

---

## 📌 Summary
Euro currency option is not available in the "Expected Salary" section.

---

## 🧪 Environment
- Application: Huntd (Web)
- Browser: Chrome 145.0.7632.76

---

## 🔁 Steps to Reproduce
1. Open the main page
2. Navigate to the "Candidates" section
3. Open the "Expectations" tab
4. Check available currency options

---

## ❌ Actual Result
Euro currency option is not displayed.

---

## ✅ Expected Result
User should be able to select both Dollar and Euro currency options.
