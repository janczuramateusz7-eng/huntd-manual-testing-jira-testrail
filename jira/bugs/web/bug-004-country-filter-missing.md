# Bug 004 – Country filter option missing in Candidate Location filter

---

## 📌 Summary
Country option is missing in the Candidate Location filter. Users can filter candidates only by City.

---

## 🧪 Environment
- Application: Huntd (Web)
- Browser: Chrome 145.0.7632.76

---

## 🔁 Steps to Reproduce
1. Open the main page
2. Go to Sign Up section
3. Fill all required fields
4. Create account as Candidate
5. Navigate to Candidates section
6. Open filters
7. Check Location filter options

---

## ❌ Actual Result
Location filter allows filtering only by City. Country option is not available.

---

## ✅ Expected Result
User should be able to filter candidates by both Country and City.

---

## ⚠️ Severity
Medium

---

## 🚨 Priority
Medium
